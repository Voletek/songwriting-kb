# Songwriting Knowledge Base

Professional songwriting knowledge base, Suno AI formatting system, and Kiro agent configuration.

## Contents

- `.kiro/steering/songwriting.md` — Auto-loaded rules for Kiro songwriting sessions
- `.kiro/agents/songwriter.md` — Reusable songwriter agent definition
- `songs/` — Song development files (raw lyrics, Suno-ready versions, production notes)

## Suno Formatting System

This repo documents a battle-tested Suno AI prompt engineering system with 5 formatting modes:

| Mode | Use Case |
|---|---|
| **A** — Per-Line Duet Tags | Intimate cinematic dialogue |
| **B** — Parenthetical () | Space-efficient call-and-response |
| **C** — Theatrical/Concept | Multi-layer concept albums |
| **D** — Character Performance | Fantasy characters, accents |
| **E** — Solo Anthem/Intimate | Accessible solo tracks |

## Setup

Clone this repo and the `.kiro/` directory will automatically configure Kiro as your professional songwriting co-writer.
